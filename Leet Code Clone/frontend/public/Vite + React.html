Write a program that reads an integer from input and prints 'Even' if it is even, and 'Odd' if it is odd.

Examples:

Input:
4
Output:
Even
Explanation:

4 is divisible by 2, so the output is 'Even'.

Input:
7
Output:
Odd
Explanation:

7 is not divisible by 2, so the output is 'Odd'.

Constraints:

-10^9 ≤ n ≤ 10^9

function checkEvenOdd(n) {
    if (n % 2 === 0) {
        return "Even";
    } else {
        return "Odd";
    }
}
